About This Platform

Precode-Collab description outlines a three-tool platform with a dark-themed UI for Perchance developers:  

### Core Features  
1. Code Compare  
   - *Current*: Manual copy/paste from two generator URLs into side-by-side text boxes.  
   - *Future*: Auto-import code directly from URLs (no manual copying).  

2. Search Engine  
   - Scans Perchance for generators tagged #precodecollab in titles/descriptions.  
   - *Exclusively* lists projects where creators *want* collaborators.  

3. Generator Chat Rooms  
   - Clicking a generator opens a live chat room *instead* of the generator itself.  
   - Enables real-time discussion *while* viewing/comparing code in the same window.  

### UI/UX Details  
- Header Bar: Quick navigation to Search and Code Compare tools.  
- Side Panel: Pop-out window with:  
  - Explanations of Precode Collab’s purpose.  
  - User feedback form for feature requests.  
- Design: Professional dark theme with high readability (optimized fonts/contrast).

Perchance.org **Terms of Service** check

Based on Perchance.org’s **Terms of Service** and community practices:

### ✅ Likely Allowed:
- **Code Compare**: Fetching public generator code via `/api/getGenerator` is acceptable.
- **Search Engine**: Scanning for `#precodecollab` tags is fine if done responsibly (rate-limited, non-intrusive).
- **UI/UX Features**: Header, side panel, and dark theme are permitted.

### ⚠️ Potential Issues:
- **Auto-import (Future Feature)**: Heavy or automated fetching may be seen as **abuse** if it overloads servers.
- **Generator Chat Rooms**: Redirecting generator clicks to chat rooms **may violate** ToS if it alters 
  intended behavior or misleads users. 

### Key Rules from ToS:
- Perchance reserves the right to **disable any content or access** that burdens systems or violates intent. 
- No explicit ban on APIs, but **excessive scraping/automation** can be restricted. 

### Recommendation:
- Implement **rate limiting** and caching.
- Make chat rooms **opt-in**, not a replacement for generator links.
- Monitor for official API updates or policies.

"Make chat rooms opt-in, not a replacement for generator links" means:

Your app should **not intercept or redirect** clicks on generator links (e.g., turning `perchance.org/my-gen` 
into a chat room). Doing so alters the intended behavior of Perchance content and may violate their 
**Terms of Service** by misrepresenting or modifying user-generated links. 

Instead:
- **Show generator links as-is**
- Provide a **separate, clear option** (e.g., a "Discuss This Generator" button) to open a chat room
- Let users **choose** to join a chat, rather than forcing it

This respects Perchance’s ecosystem, avoids confusion, and complies with their policy 
against unauthorized modification of generator experiences.
